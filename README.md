# Uxpload

## Basic deployment:

Install node and redis:

    http://nodejs.org/
    http://redis.io/download

Clone the repo:

    git clone git@github.com:jonfm/uxpload.git
    cd uxpload

Install node dependencies:

    npm install

Run the server:

    node server.js

Then hit http://localhost:3000/ from your browser...
